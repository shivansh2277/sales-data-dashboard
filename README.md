# 📊 Sales Data Cleaning & Dashboard Project

## 🚀 Overview
This project demonstrates an end-to-end data analysis workflow:
- Cleaning messy sales data
- Fixing inconsistencies and missing values
- Performing business analysis
- Building a visual dashboard

---

## ❗ Problem
Raw business data is often:
- Incomplete (missing values)
- Incorrect (wrong calculations)
- Duplicated
- Hard to interpret

This makes decision-making difficult.

---

## ✅ Solution
I built a Python-based pipeline to:
- Clean and validate the dataset
- Correct revenue calculations
- Standardize formats
- Generate meaningful business insights
- Visualize data through a dashboard

---

---

## 🔍 Key Insights
- Revenue shows strong seasonal peaks in Q4
- Electronics dominate sales → dependency risk
- Some cities consistently underperform

---

## 📈 Business Recommendations
- Diversify product categories
- Increase focus on low-performing regions
- Optimize inventory for peak months

---

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- NumPy
